# DevNest v0.1.2

**Windows desktop release** — reliable in-app updates and clearer onboarding.

## Download

- **Installer:** [DevNest_0.1.2_x64-setup.exe](https://github.com/khayson/DevNest/releases/download/v0.1.2/DevNest_0.1.2_x64-setup.exe)
- **All assets:** [GitHub Releases v0.1.2](https://github.com/khayson/DevNest/releases/tag/v0.1.2)

## What's new

- **Fixed update installs** — installer stops background `devnest.exe` before replacing files (no more “Error opening file for writing”)
- **Smarter in-app updates** — services stop automatically before the installer runs
- **Onboarding Activity log** — see bootstrap, install, and error messages in the setup wizard
- **Toast notifications** during setup so nothing fails silently

## Upgrading

1. **About → Check for updates** (recommended on v0.1.1+), or
2. Download and run the new installer

If an update still fails, quit DevNest fully and end any `devnest.exe` in Task Manager, then retry.
