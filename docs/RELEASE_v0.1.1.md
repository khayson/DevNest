# DevNest v0.1.1

**Windows desktop release** — onboarding fixes and community website.

## Download

- **Installer:** [DevNest_0.1.1_x64-setup.exe](https://github.com/khayson/DevNest/releases/download/v0.1.1/DevNest_0.1.1_x64-setup.exe)
- **All assets:** [GitHub Releases v0.1.1](https://github.com/khayson/DevNest/releases/tag/v0.1.1)
- **Website:** [devnest.dev](https://devnest.dev) (Vercel — set your domain after deploy)

## What's new

- **Faster onboarding** — setup wizard appears right away; no more long blank loading screen
- **Stuck on Environment?** — use *Continue without connection* after 12 seconds
- **Cleaner UI** — removed sparkle/AI-style icons; DevNest branding on setup
- **Community** — report bugs and suggest features from the website or GitHub

## Upgrading from v0.1.0

1. Open DevNest → **About → Check for updates**, or
2. Download and run the new installer (overwrites in place)
