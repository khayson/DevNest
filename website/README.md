# DevNest website

Marketing site for [DevNest](https://github.com/khayson/DevNest) — download, changelog, bug reports, and feature requests.

## Deploy on Vercel

1. Import the repo in [Vercel](https://vercel.com/new)
2. Set **Root Directory** to `website`
3. Deploy (Vercel detects Vite via `vercel.json`)

Optional custom domain: `devnest.dev` or `getdevnest.com` → add in Vercel project settings.

## Local dev

```bash
cd website
npm install
npm run dev
```

## Links

- Download uses GitHub Releases API for the latest Windows installer
- Bug reports → GitHub Issues (`bug_report` template)
- Features → GitHub Issues (`feature_request` template)
- Reviews / Q&A → GitHub Discussions
